FAQ

Questions?
Answered.

Is Whispr really free?

Yes. Whispr is open source and completely free. There are no subscriptions, no premium tiers, no time-limited trials, and no hidden costs. The offline models are included — download once and use forever.

Does it work without internet?

Yes. Three of the four models (Parakeet v3, Whisper Large Turbo, Whisper Small) run entirely on your Mac. No internet connection needed after the initial model download. The fourth option (OpenAI gpt-4o) requires internet and your own API key, but it’s completely optional.

How accurate is the transcription?

Very accurate for English. We recommend Parakeet v3 for the best balance of speed and accuracy — it runs on Apple’s Neural Engine and handles natural speech well. For maximum accuracy, the OpenAI gpt-4o cloud model is available with your own API key.

What about other languages?

Currently, Whispr is optimized for English. The Whisper Large Turbo model has multilingual capabilities, but the app’s cleanup and filler-word removal are tuned for English. Multi-language support is on the roadmap.

How do I set up Whispr?

Open the DMG, drag Whispr to Applications, and launch it. macOS will ask you to grant Microphone and Accessibility permissions (needed for the global hotkey and paste). That’s it — hold fn and start speaking.

Why does macOS say it can’t verify Whispr?

Whispr isn’t notarized with Apple (that requires a $99/year developer subscription). It’s completely safe — fully open source and auditable. See the 3-step fix →

Is it safe? Can I trust it?

Whispr is fully open source — you can inspect every line of code on GitHub. Audio is processed on-device and never transmitted. The app requires Microphone and Accessibility permissions (for the global hotkey and paste), both of which macOS explicitly asks you to grant.

Fix “Apple could not verify”

Whispr is open source and safe, but since it’s not notarized with Apple, macOS blocks it on first launch. Follow these three steps:

1

Dismiss the error

When you first open Whispr, you’ll see this dialog. Click Done.

2

Open Privacy & Security settings

Go to System Settings → Privacy & Security. Scroll to the bottom. You’ll see “Whispr was blocked to protect your Mac.” Click Open Anyway.

3

Confirm and launch

macOS will ask one more time. Click Open Anyway. Whispr will launch normally from now on.

Alternative: Run this in Terminal instead: xattr -cr /Applications/Whispr.app
